Props(properties):

* It is parameter of functional parameter which is used to pass data inside component.
* props return data in object format.  
* Accessing props in function Component:
   Syntax: props.propsname
* Accessing props in class component:
   syntax: this.props.propsname

eg: const FunctionComp=(props){
	return(
         <div>
         <p>my name is {props.Name}</p>
	</div>
	)
}



Destructuring of Props:
*To avoid writing props. everytime to access props data, we use this feature
  syntax: const {name,age,gender}=props;


State:

* In React class component, properties should be keep in an object called state.
* It is used to handle essential in class component.
* It returns object  type of data.
* In class components we write state inside the constructor

syntax: eg:   class Component extends Component
	      {
		 constructor(props){
		  super(props)
	          this.state={
		   name:"selva"
  		   age:"22"	
		}
		render(){
	        return(
		)		
			}  
		
	      }

Note: To use this property in function comp, we have hooks called (useStateHooks).
