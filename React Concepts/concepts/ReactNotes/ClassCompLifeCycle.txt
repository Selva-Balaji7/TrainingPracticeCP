Class Components Life-Cycle Methods:

  * It comes with four phases

  1.Mounting Phase: 

      *  When an instance of a Component is being created and inserted into the DOM.
      *  It contains: constructor, static getDerivedStateFromProps, render and ComponentDidMount.
      	
	constructor:   * When class object initiated,constructor get executed automatically.

	static getDerivedStateFromprops(props,state): * It is a static method which execute automatically, and is used to
	set values coming from props to state.
    
	render: * It is responsible to render component data.

	componentDidMount: * It executed immediately after render method. 
   
  2.Updating Phase: 
     * When a component is being re-rendered as a result of changes to either its props or state.
     * It contains: static getDerivedStateFromProps, shouldComponentUpdate, render, getSnapshotBeforeUpdate and componentDidUpdate.

  3.Unmounting phase:  
     * when a component is being removed from the DOM.
     * It contains: componentwillUnmount
     
  4.Error phase:
    * when there is an error during rendering, in a lifecycle method, or in the constructor of any child component.
    * It contains: static getDerivedStateFromError and componentDidCatch
